### Resize Image
This is python script to resize the image in the directory by % value.
Image of resolution = 720 * 480 
Ratio in Percentage = 50 % 
NewImage Resolution = 360 * 240

### Note:
It blurrs out pixels since the resolution has changed but overall image effect remains same.

### Usage: 
$./image_resize.py heart.png 5
Resize image 'heart.png' by 5%
It will store the result in the same directory resized_heart.png
